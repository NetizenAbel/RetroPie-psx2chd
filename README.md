# RetroPie PSX2CHD

A tool to compress PSX games into CHD format.

## Installation

```
cd /home/pi/
git clone https://github.com/kashaiahyah85/RetroPie-psx2chd
cd RetroPie-psx2chd
sudo chmod +x psx2chd.sh
```

## Updating

```
cd /home/pi/RetroPie-psx2chd/
git pull
```

## Usage

```
./psx2chd.sh
```
